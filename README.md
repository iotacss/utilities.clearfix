# Clearfix Utility #

The clearfix utility contains a simple clearfix helper class and placeholder.


### Installation ###

```
npm install --save iotacss-clearfix
```


### Options ###

```sass
$iota-utils-clearfix-namespace  : 'cf' !default;
```


### Classes ###

```css
.u-cf
```
